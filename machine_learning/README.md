# Machine Learning

It seperates to two titles:

**Supervised** and **Unsupervised**

## Supervised:

- Task driven
- Pre categorized data
- Models:
    - Classification
    - Regression
    - and more...

## Unsupervised:

- Data driven
- Unlabelled data
- Models:
    - Clustering
    - Association
    - Dimensionality Reduction
    - and more...
